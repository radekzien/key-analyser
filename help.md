# How to use the Key Analyser
1. Press the 'Open File' Button
2. Find the audio file (.mp3 or .wav) that you wish to analyse.
3. Once uploaded, press the Analyse button.
4. Once analysed, the large text shows you the detected key.

### Doesn't sound right? Look at the related keys to find the related (Subdominant, Dominant, or relative Major/minor) keys.
